# Vamos Academy — Spanish Landing Page (React)

موقع هبوط لأكاديمية Vamos لتعليم الإسبانية، مبني بـ React + Vite.

---

## 🚀 تشغيل سريع | Quick Start

```bash
# 1. تنصيب المكتبات
npm install

# 2. تشغيل وضع التطوير (للمعاينة على جهازك)
npm run dev
# يفتح على http://localhost:5173

# 3. بناء الإصدار النهائي للنشر
npm run build
# الناتج هيكون في مجلد dist/
```

---

## ⚙️ الإعدادات | Configuration

كل الإعدادات في ملف واحد: **`src/config.js`**

### 1️⃣ رقم الواتساب
موجود بالفعل: `201064298604`. غيّره لو احتجت.

### 2️⃣ إعداد إرسال الفورم (مهم!)

الفورم بيرسل البيانات في الخلفية **بدون فتح واتساب على الموبايل**.
عشان يشتغل، لازم تختار طريقة واحدة (أو الاتنين معاً للأمان):

#### الطريقة الأسهل والأسرع: **CallMeBot** (واتساب مباشر)

1. ضيف هذا الرقم لجهات اتصال الواتساب: **+34 644 51 95 23**
2. ابعتله رسالة بالضبط كده:
   ```
   I allow callmebot to send me messages
   ```
3. هيبعتلك رد فيه API key (رقم زي `1234567`)
4. افتح `src/config.js` وحط الـ key:
   ```js
   callMeBotApiKey: '1234567', // ← هنا
   ```
5. خلاص! أي طلب يدخل الفورم رح يوصلك على واتساب مباشرة.

> ⚠️ **ملاحظة:** CallMeBot خدمة مجانية فيها rate limit (رسالة كل بضع ثوان). 
> ممتاز للأكاديميات الصغيرة، لو حجم الطلبات كبير جداً فكّر في WhatsApp Business API.

#### الطريقة البديلة: **Web3Forms** (إيميل)

1. روح على https://web3forms.com
2. اكتب إيميلك، هيبعتلك access key
3. افتح `src/config.js` وحطه:
   ```js
   web3FormsAccessKey: 'YOUR-KEY-HERE',
   ```
4. أي طلب رح يوصلك على إيميلك. ممكن بعدين تربط الإيميل بالواتساب عبر Zapier.

#### الأكثر أماناً: **اعمل الاتنين**

حط الـ key للاتنين. الفورم هيرسل عبر الـ 2 في نفس الوقت، فلو واحد فشل التاني يلحق.

## 📦 النشر | Deploying

بعد `npm run build`، عندك مجلد `dist/` فيه كل اللي تحتاجه.

### خيارات مجانية للنشر:

- **Netlify** (الأسهل): اسحب مجلد `dist/` لـ https://app.netlify.com/drop
- **Vercel**: `npm i -g vercel && vercel`
- **GitHub Pages**: ارفع المجلد لـ branch اسمه `gh-pages`
- **Cloudflare Pages**: ربط GitHub repo

كل واحد منهم يديك دومين مجاني (مثلاً `vamos-academy.netlify.app`)، 
وبعدين تقدر تربطه بالدومين بتاعك.

## 🌍 إضافة لغات جديدة

في المستقبل لو حبيت تضيف صفحة للفرنسي أو الإيطالي:

1. **انسخ مجلد المشروع**
2. غيّر محتوى `src/data.js` (الكورسات والتقييمات)
3. غيّر النصوص في `src/App.jsx` (العناوين والوصف)
4. غيّر `src/config.js` (لو في رقم/key مختلف)

## 📁 هيكل المشروع

```
src/
├── config.js          ← الإعدادات (رقم الواتساب، API keys، تاريخ الورشة)
├── data.js            ← بيانات الكورسات والتقييمات
├── submit.js          ← منطق إرسال الفورم في الخلفية
├── App.jsx            ← المكوّن الرئيسي
├── CourseCard.jsx     ← كارت الكورس
├── ContactForm.jsx    ← الفورم
├── icons.jsx          ← أيقونات SVG
├── styles.css         ← التصميم كله
└── main.jsx           ← نقطة دخول React
```

## ✅ المزايا المضمنة

- 📱 **Fully responsive** — يشتغل من 320px لـ 1920px
- ⏱️ **Live countdown** للورشة المجانية
- 🎯 **Form pre-fill** — لما تضغط "احجز" في أي كورس، الفورم يتفتح بالكورس مظبوط
- 🔄 **Background submission** — مفيش redirect لواتساب، البيانات بترسل بدون تشتيت العميل
- 🌐 **RTL support** كامل
- ⚡ **Vite** — بناء سريع، load أسرع
- 🎨 **نفس التصميم** المعتمد، بدون أي تغيير

## 🆘 مشاكل شائعة

**الفورم بيرجع "حدث خطأ":**
- تأكد إنك حطيت الـ API key في `src/config.js`
- جرب الـ Web3Forms كبديل لو CallMeBot مش شغّال

**CallMeBot مش بيوصل رسايل:**
- تأكد إنك بعتت رسالة "I allow callmebot to send me messages" للرقم +34 644 51 95 23
- استنى ثواني بين كل رسالة (rate limit)

**Countdown مش بيتحدث:**
- تأكد إن تاريخ الورشة في `src/config.js` بالصيغة الصحيحة (ISO 8601)
